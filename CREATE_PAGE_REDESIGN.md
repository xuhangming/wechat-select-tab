# 创建抽签页面重新设计说明

## 更新内容

已将创建抽签页面重新设计为更清晰的分栏布局，提升用户体验和功能完整性。

## 新布局结构

### 1. 抽签基本信息
- **抽签标题**（必填）：最多50个字符
- **抽签描述**（选填）：最多200个字符，支持多行输入，实时显示字符计数

### 2. 抽签选项设置
- **选项列表**：最多添加1000个选项
- **选项编号**：自动显示序号（圆形紫色标签）
- **选项名称**：输入选项内容
- **选项数量**：设置每个选项的名额数量（默认为1）
- **删除按钮**：红色圆形按钮，至少保留一个选项
- **添加按钮**：虚线边框按钮，显示当前选项数量（如：添加选项 (3/1000)）

### 3. 时间设置
- **按天设置**：使用日期选择器，简化时间选择
- **开始日期**（必填）：默认为当前日期
- **结束日期**（必填）：默认为明天
- **持续时间显示**：自动计算并显示持续天数（蓝色提示框）
- **时间范围**：开始时间为当天00:00:00，结束时间为当天23:59:59

### 4. 高级设置
- **抽签口令**（选填）：最多20个字符，设置后参与者需要输入口令才能抽签
- **抽签结果查看权限**：
  - 所有人可见（默认）
  - 仅参与者可见
  - 仅创建者可见

## UI 设计特点

### 分区设计
- 每个功能区域独立显示，背景为白色卡片
- 区域标题加粗显示，字号32rpx
- 区域描述使用灰色小字，提供额外说明

### 表单元素
- 输入框：浅灰色背景，圆角设计
- 选择器：与输入框样式一致
- 文本域：支持自动高度调整
- 单选框：卡片式布局，浅灰色背景

### 选项列表
- 圆形编号标签：紫色渐变背景
- 输入框布局：名称占2/3宽度，数量占1/3宽度
- 删除按钮：红色圆形，仅在多个选项时显示
- 添加按钮：虚线边框，显示当前数量

### 提交按钮
- 渐变紫色背景（#667eea → #764ba2）
- 圆角44rpx，带阴影效果
- 加载状态时颜色变浅

## 功能改进

### 1. 时间设置简化
- 从复杂的多列选择器改为简单的日期选择器
- 自动设置时间为全天（00:00:00 - 23:59:59）
- 实时显示持续天数

### 2. 选项管理优化
- 支持最多1000个选项（原来无限制）
- 显示当前选项数量
- 每个选项独立编号
- 删除按钮更直观

### 3. 新增功能
- 抽签描述字段
- 抽签口令功能
- 结果查看权限控制
- 字符计数显示

### 4. 验证优化
- 独立的验证逻辑
- 更清晰的错误提示
- 实时清除错误状态

## 用户体验提升

1. **更清晰的布局**：分区设计让功能一目了然
2. **更简单的操作**：日期选择器比多列选择器更易用
3. **更直观的反馈**：实时显示字符计数和持续天数
4. **更强大的功能**：支持描述、口令和权限控制
5. **更美观的界面**：现代化的卡片式设计

## 注意事项

1. 选项数量限制为1000个，超过会提示
2. 至少需要一个选项才能创建
3. 结束日期必须晚于开始日期
4. 口令为选填，留空表示无需口令
5. 默认结果查看权限为"所有人可见"
