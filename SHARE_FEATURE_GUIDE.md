# 活动分享功能使用指南

## 功能概述

活动分享功能允许抽签活动的创建者将活动分享给其他用户，其他用户通过分享链接可以直接进入活动详情页参与抽签。

## 功能特性

✅ **创建者专属** - 只有活动创建者可以看到分享按钮  
✅ **两种分享方式** - 支持分享给好友和分享到朋友圈  
✅ **自动跳转** - 接收者点击分享链接自动进入活动详情页  
✅ **完整信息** - 分享卡片显示活动标题，吸引参与

## 使用方法

### 方式一：使用页面内分享按钮（推荐）

1. **进入活动详情页**
   - 创建活动后会自动跳转到详情页
   - 或从历史记录中点击活动进入

2. **点击分享按钮**
   - 在活动基本信息区域，可以看到"📤 分享活动"按钮
   - 点击按钮会弹出微信分享菜单

3. **选择分享对象**
   - 可以选择分享给微信好友
   - 或分享到微信群

### 方式二：使用右上角分享

1. **进入活动详情页**
   - 任何用户都可以使用此方式分享

2. **点击右上角"..."菜单**
   - 选择"转发"选项

3. **选择分享对象**
   - 分享给好友或群聊

### 方式三：分享到朋友圈（需要支持）

1. **进入活动详情页**

2. **点击右上角"..."菜单**
   - 选择"分享到朋友圈"选项

3. **编辑分享内容**
   - 可以添加自己的评论
   - 发布到朋友圈

## 接收者体验

### 1. 收到分享

接收者会看到分享卡片，显示：
- 标题：邀请你参与抽签：[活动名称]
- 小程序名称和图标

### 2. 点击进入

点击分享卡片后：
- 自动打开小程序
- 直接进入该活动的详情页面
- 可以查看活动信息和参与者列表

### 3. 参与抽签

如果满足参与条件：
- 活动正在进行中
- 还未参与过
- 有剩余名额

则可以点击"参与抽签"按钮进行抽签。

## 分享内容说明

### 分享标题格式

```
邀请你参与抽签：[活动标题]
```

例如：
- 邀请你参与抽签：周末值班安排
- 邀请你参与抽签：团建活动分组

### 分享路径

```
/pages/detail/detail?id=[活动ID]
```

系统会自动携带活动ID，确保接收者能正确访问活动。

## 技术实现

### 页面配置

在 `pages/detail/detail.js` 中实现了两个分享方法：

```javascript
// 分享给好友
onShareAppMessage() {
  return {
    title: `邀请你参与抽签：${event.title}`,
    path: `/pages/detail/detail?id=${eventId}`,
    imageUrl: '' // 可自定义分享图片
  };
}

// 分享到朋友圈
onShareTimeline() {
  return {
    title: `邀请你参与抽签：${event.title}`,
    query: `id=${eventId}`,
    imageUrl: '' // 可自定义分享图片
  };
}
```

### UI 组件

在 `pages/detail/detail.wxml` 中添加了分享按钮：

```xml
<button class="share-btn" open-type="share">
  <text class="share-icon">📤</text>
  <text>分享活动</text>
</button>
```

## 注意事项

### 1. 权限控制

- ✅ 分享按钮仅对创建者显示
- ✅ 但任何用户都可以通过右上角菜单分享
- ✅ 接收者需要满足参与条件才能参与抽签

### 2. 活动状态

- 如果活动未开始，接收者会看到"活动未开始"提示
- 如果活动已结束，接收者会看到"活动已结束"提示
- 只有活动进行中才能参与抽签

### 3. 重复参与

- 系统会自动检查用户是否已参与
- 已参与的用户会看到"已参与"状态
- 无法重复参与同一活动

### 4. 名额限制

- 如果所有类目名额已满，会提示"抽签名额已满"
- 接收者仍可查看活动详情和参与者列表

## 自定义选项

### 自定义分享图片

可以在分享配置中设置 `imageUrl` 参数：

```javascript
onShareAppMessage() {
  return {
    title: `邀请你参与抽签：${event.title}`,
    path: `/pages/detail/detail?id=${eventId}`,
    imageUrl: '/images/share-cover.png' // 自定义分享图片
  };
}
```

图片要求：
- 尺寸：5:4 比例
- 格式：JPG 或 PNG
- 大小：不超过 128KB

### 自定义分享标题

可以修改分享标题的格式：

```javascript
title: `${event.title} - 快来参与抽签吧！`
```

## 常见问题

### Q1: 为什么我看不到分享按钮？

A: 分享按钮只对活动创建者显示。如果你不是创建者，可以使用右上角的"..."菜单进行分享。

### Q2: 分享后对方无法打开？

A: 请确保：
- 小程序已发布或在体验版中
- 对方的微信版本支持小程序
- 活动ID正确传递

### Q3: 可以分享到其他平台吗？

A: 目前只支持微信内分享（好友、群聊、朋友圈）。微信小程序不支持分享到其他平台。

### Q4: 分享链接会过期吗？

A: 分享链接不会过期，但活动本身有时间限制。如果活动已结束，接收者仍可查看详情，但无法参与抽签。
