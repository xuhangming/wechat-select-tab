# 抽签详情页样式优化

## 优化时间
2026-03-03

## 优化目标
参考提供的抽签助手页面布局，对抽签详情页进行样式优化，保持原有功能不变。

## 优化内容

### 1. 整体布局优化
- **背景色**: 从 `#f5f5f5` 改为 `#f8f8f8`，更加柔和
- **内边距**: 添加 `20rpx` 的容器内边距，增加呼吸感
- **底部间距**: 增加到 `140rpx`，为底部按钮留出更多空间

### 2. 活动信息卡片重设计
**原设计**：
- 简单的白色背景区域
- 垂直排列的标题、状态、时间信息
- 使用文本标签显示状态

**新设计**：
- 圆角卡片设计（`border-radius: 16rpx`）
- 添加轻微阴影（`box-shadow: 0 2rpx 12rpx rgba(0, 0, 0, 0.04)`）
- 标题更大更醒目（`40rpx`，加粗）
- 状态使用彩色标签（badge）显示：
  - 未开始/进行中：蓝色背景 `#e6f7ff`，蓝色文字 `#1890ff`
  - 已结束：灰色背景 `#f5f5f5`，灰色文字 `#999`
- 新增进度标签：显示"已抽 X/Y"，绿色主题
- 移除详细时间显示，界面更简洁

### 3. 类目列表重设计
**原设计**：
- 单个白色背景区域包含所有类目
- 类目之间用分隔线分隔
- 显示"总数"和"剩余"两个数值

**新设计**：
- 每个类目独立的卡片（`border-radius: 16rpx`）
- 卡片之间有 `16rpx` 间距
- 类目信息水平排列：名称 + 数量（× N）
- 已抽满的类目：
  - 使用渐变背景高亮（`linear-gradient(135deg, #f0f9ff 0%, #e6f7ff 100%)`）
  - 右侧显示绿色"已抽满"标签
- 添加过渡动画效果（`transition: all 0.3s ease`）

### 4. 底部按钮优化
**原设计**：
- 白色背景的固定底栏
- 紫色渐变按钮（`#667eea → #764ba2`）
- 有顶部边框和阴影

**新设计**：
- 透明背景，无边框
- 绿色渐变按钮（`#52c41a → #73d13d`）
- 更大的按钮高度（`96rpx`）
- 更大的圆角（`48rpx`）
- 添加绿色阴影效果（`box-shadow: 0 8rpx 24rpx rgba(82, 196, 26, 0.3)`）
- 禁用状态移除阴影

### 5. 数据计算增强
在 JavaScript 中新增：
- `totalCount`: 所有类目的总数量
- `assignedCount`: 已分配的数量
- 用于显示进度标签"已抽 X/Y"
